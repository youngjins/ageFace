자바 제어문

* 제어문
- 프로그램의 흐름에 영향을 주고 때에 따라 제어가 가능하도록 하는 것
- 제어문의 종류는 조건문과 반복문이 있다.

* 조건문(if문, switch문)
- 조건문은 주어진 조건식의 결과에 따라 별도의 명령을 수행하도록 제어하는 명령문

* 반복문(for문, while문, do~while문)
- 반복문이란 프로그램 내에서 똑같은 명령을 일정 횟수만큼 반복하여 수행하도록 제어하는 명령문
- 동일하거나 유사한 문장을 수치에 의해 제어하는 문.


조건문

* if문
- 조건식과 하나의 문자 또는 블록{}으로 구성.
- if(조건식) 실행문;
- if(조건식){ 실행문; }else{ 실행문; }
- if(조건식){ 실행문; } else if(조건식){ 실행문; } else{ 실행문; }

* switch문
- 특정 수나 문자에 의해 위치 이동을 제어
- if 문과 달리 조건식의 계산 결과가 int 범위 이하의 정수만 가능.
- java7 이후 부터는 문자열도 가능
- case 문의 값으로 변수를 사용할 수 없다. (리터럴 또는 상수만 가능)
- switch(변수){ case 값1: 실행문; break; case 값2: 실행문; break; default : 만족하는 case가 없을 때 실행. 생략가능}


반복문

* for문
- 동일하거나 유사한 문장을 수치에 의해 제어하는 문
- for(①초기화식; ②조건식; ④증감식){ // 조건식이 true일때 반복 실행이 될 내용③실행문;}

* continue문
- 자신이 포함된 반복문의 끝으로 이동한다.
- continue문 이후의 문장들은 수행되지 않는다.

* break문
- 자신이 포함된 하나의 바복문 또는 switch문을 빠져 나온다.

* while문
- for문과 같이 문장의 반복 수행을 하나 for문이 수치에 의한 반복이라면 while문은 행위에 의한 반복문이다.
- while(①조건식){ // 조건식이 true일때 반복 실행이 될 내용 ②실행문;}

* do ~ while문
- 선 실행 후 조건 비교 반복하는 제어문.
- 조건식에 의해 반복 실행한다는 점에서는 while문과 동일
- do{ // 조건식이 true일때 반복 실행이 될 내용 ①실행문;} while(②조건식);
